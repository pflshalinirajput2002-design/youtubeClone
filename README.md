# youtubeClone
Responsive You tube clone using -HTML5 – Page structure &amp; video elements  CSS3 – Responsive layout and modern UI styling  JavaScript – Page navigation &amp; dynamic user interactions 
This is a Video Clone Web Application built using HTML, CSS, and JavaScript. The project is inspired by modern video streaming platforms, providing a clean UI and smooth navigation between video listing and video playing pages.
The project includes two main pages:
index.html — Displays video thumbnails with title and channel information

play-video.html —  Plays the first thumbnail video and shows related video suggestions on the sidebar
